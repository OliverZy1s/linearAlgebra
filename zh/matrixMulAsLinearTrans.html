# 矩阵乘法作为线性变换的形式理解

# 矩阵乘法

在上一节里我们介绍了线性变换是如何从几何上定义的以及如何理解矩阵去描述一个线性变换的。 当理解了这一点之后，我们可以再思考一个问题：

# 问题❓：如果要经过多个线性变换，我们应当怎么处理？

就是一个向量先经过线性变换A，然后再经过线性变换B，这种过程称为线性变换的复合。 经过这两次线性变换后我们肯定可以得到变换后的基向量，而这个基向量将描述复合的线性变换。

除了这种理解方法，我们还可以先求出向量经过线性变换A得到的新向量，再把这个新向量经过线性变换B，就可以的得到最后的结果。 这两种方法所最后得到的结果是一样的，我们可以表示出来看这两种方法的关联：

# 结合律

十分类似的问题：

问题❓：矩阵乘法满足结合律吗？

也就是说，我们先经过线性变换C，B然后A，跟经过线性变换C，然后B，A，有什么区别吗？

答案是：没有，从把矩阵看成线性变换的复合这一角度上，这个结论也是十分自然的。
