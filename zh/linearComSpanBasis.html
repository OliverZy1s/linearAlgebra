# 线性组合，张成的空间和基

# 基向量

在上一节学过了向量坐标这个概念之后，我们还有一种很有趣的方法来看待这些坐标，这个对线性代数非常重要。 再回过头来看用坐标表示一个向量，其实也可以这样理解：

我们先给出两个特殊的向量 i⃗=(1,0)\vec{i}=(1,0)​i​⃗​​=(1,0) 和 j⃗=(0,1)\vec{j}=(0,1)​j​⃗​​=(0,1)，而原来的向量v⃗=(3,−2)\vec{v}=(3,-2)​v​⃗​​=(3,−2)就可以理解为 i⃗\vec{i}​i​⃗​​ 乘上x方向上的标量再加上 j⃗\vec{j}​j​⃗​​ 乘上y方向上的标量，也就是 3i⃗−2j⃗3\vec{i}-2\vec{j}3​i​⃗​​−2​j​⃗​​。 这种将一个向量通过一组特殊的向量经过数乘再相加表示的思想是非常重要的。

而这两个向量就一起被称为坐标系的一组基

补充

i⃗\vec{i}​i​⃗​​ 和 j⃗\vec{j}​j​⃗​​ 也有一个特殊的名称，i⃗\vec{i}​i​⃗​​ 被称为是x方向上的单位向量， j⃗\vec{j}​j​⃗​​ 被称为是y方向上的单位向量。

# 问题❓：我们能不能选择不同的基向量？

其实是可以的，只要我们选择的基向量能够通过数乘和相加表示完整的坐标系， 那么这一组向量跟也可以跟 i⃗\vec{i}​i​⃗​​, i⃗\vec{i}​i​⃗​​ 一样作为一组基向量。

那么，又有一个新的问题，当我们确定了一组基向量之后，通过数乘和相加这两个运算， 我们能得到哪些二维向量？

答案是：在二维平面上，通过改变基向量乘的标量，我们能表示整个平面的向量。

我们希望你学到这里起码能接受一点，即每当我们用数字描述向量时，这其实依赖着我们使用的基。

# 线性组合

我们将基向量乘上一个标量后相加的行为称作是，线性组合：

  • Def. v⃗\vec{v}​v​⃗​​ 和 w⃗\vec{w}​w​⃗​​ 的线性组合定义为：av⃗\vec{v}​v​⃗​​+bw⃗\vec{w}​w​⃗​​，其中a,b都是常量。

# 张成的空间

一组初始向量进行任意的线性变换可能造成两个结果：

  • 大部分情况下，对于一对初始向量，通过线性变换可以表示平面上的任何一个向量，也就是可以触及到平面上的任意一点。

  • 但是对于一组共线的初始向量，能表示的向量终点就被限制到了一条直线上。

补充

当然，严格来讲还有第三种情况，就是两个都初始向量都是零向量，那么它们能表示的也只有零向量。

因此十分自然地，我们就可以再介绍一个术语，向量张成的空间：

  • Def. 所有可以表示为给定向量线性组合的向量集合称为给定向量张成的空间（Span）

那么刚刚谈到的几种情况用专业的语言再次叙述则是：对于大部分二维向量来说，它们张成的空间是所有二维向量的集合，但当 这对初始向量共线时，它们张成的空间就是终点落在一条直线上的向量的集合。 换句话说，两个向量的张成的空间就是这两个向量经过数乘和相加这两个运算得到的所有可能的向量的集合。 这也是为什么我们在之前会说数乘和加法这两种运算如此重要的原因。

# 将向量看成一个点

之前所有的向量在坐标轴上的表示都是用一个有向线段来表示的， 但是当聚集了很多向量时，则会显得有些杂乱。因此我们也可以用向量终点 表示该向量，但是这么表示的前提是向量的起点要是原点。

总的来说就是，当只有少数向量时，我们通常会把它想成一个有向线段，但是当面对 一些向量时，更方便的方式是把它们看成一些点。

类似的，考虑二维平面上的所有向量时，我们只需把每个向量抽象成它的终点， 这样我们就不必考虑所有的箭头，只需考虑整个二维平面就行了。

# 三维中的向量空间

问题：如果两个方向不一样的向量在三维空间中，那么它们张成的空间是怎样的？

这两个向量张成的空间就是它们所有的线性组合向量的集合，也就是缩放再相加之后所有可能得到的向量。我们可以想象到 随着线性组合中的两个标量的改变，它们所形成向量的终点形成了三维空间中一个过原点的平面，这个平面就是这两个向量张成的空间。

那么，如果我们又加了第三个向量，它们张成的空间又是什么样的呢？

和两个初始向量的情况相似，该三个向量所有可能的线性组合构成了它们张成的空间。

  • 如果第三个向量在原来两个向量所组成的那个平面中的话，那么很显然第三个向量没有做出什么贡献， 向量空间然是原来的平面。

  • 如果第三个向量不在原来两个向量所组成的那个平面中的话，那么其实这三个向量的线性组合形成的向量就能充分触及到空间的每一个角落， 也就是说这三个向量的向量空间是整个三维空间。

# 线性相关与线性无关

# 线性相关

对于：

  • 二维平面中两个向量共线
  • 三维空间中第三个向量在前两个向量组成的平面内

这两种情况情况我们需要有一个术语来描述这种："即至少有一个向量是多余的，没有对张成的空间做出任何贡献"的情况。因此我们可以给出以下定义：

  • Def. 如果有多个向量，并且可以移除其中一个或多个而不“减少”张成的空间，当这种情况发生时，称这些向量是线性相关的

另一种表述方法是，如果其中一个向量可以表示为其它向量的线性组合，因为这个向量已经落在其他向量张成的空间中，同样称这些向量是线性相关的

# 线性无关

线性无关的定义也可以类似给出：

  • Def. 如果所有向量都给张成的空间增添了新的维度，它们就被称作是线性无关的

同样的，也可以说当其中任何一个向量不能用其他向量通过线性组合表示，那么这些向量则是线性无关的。

# 基的严格定义

再回到我们最初的问题，基的定义不只是限制于 i⃗=(1,0)\vec{i}=(1,0)​i​⃗​​=(1,0) 和 j⃗=(0,1)\vec{j}=(0,1)​j​⃗​​=(0,1)，更严谨的定义是：

  • Def. 张成给定空间的一个线性无关向量集
